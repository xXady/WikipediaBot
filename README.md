# WikipediaBot

Для того, чтобы бот заработал нужно:
1) Скачать архив и разархивировать его
2) Запустить программу WikipediaBot.exe
3) Зайти в телеграм и в поиске ввести @WikipediaUrFUBot
4) Бот готов к работе
